libraries: numpy, pandas, sklearn, collections, matplotlib

scrapper.py: used for getting data from IPL official website
LaTex Files for Report: LaTex code for final report.
boundaries 4s 6s.py: boundaries fours & sixes analysis
home_away_toss.py: analysis of affect of toss & home-away ground
